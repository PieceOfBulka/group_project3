# hse_smadimo_gp3

## Запуск

```bash
python agent.py
```

# Описание
Описание
Автономный ReAct-агент на базе LangGraph и LangChain, который выполняет полный ML-пайплайн для анализа вакансий с HH.ru и предсказания зарплат. Агент самостоятельно загружает данные, проводит EDA, обучает несколько моделей, подбирает гиперпараметры и формирует HTML-отчет.

Возможности
ML-пайплайн (вкладка "ML Агент"):

Демо-парсинг HH.ru (имитация запроса к API)
Загрузка и EDA датасета с 5000 вакансий
Предобработка: целевая переменная, признаки из навыков, город, уровень позиции
Обучение трех моделей: Ridge, RandomForest, GradientBoosting
Подбор гиперпараметров лучшей модели через GridSearchCV (cv=3)
Предсказание зарплаты для заданной вакансии
HTML-отчет с метриками и историей запусков
Сравнение LLM (вкладка "Сравнение LLM"):

Тестирование 8 LLM-моделей по трем задачам (EDA, предобработка, обучение). Метрики: tasks_passed, retries, time, MAE, R2, success_rate_pct. Результаты сохраняются в llm_comparison.csv.

Интерфейс сохранения модели:

Таблица с метриками всех трех моделей
Выбор модели для сохранения
Сохранение в .pkl по заданному пути
Скачивание .pkl через браузер
Экспорт воркфлоу агента:

Кнопка "Экспорт воркфлоу" генерирует agent_workflow.json с описанием архитектуры агента, инструментов, памяти и техник промптинга.

Архитектура агента
Тип: ReAct (Reasoning + Acting)

Техники промптинга:

Chain-of-Thought: агент объясняет рассуждение перед каждым действием
Few-shot prompting: примеры корректных шагов в system prompt
Contrastive prompting: примеры ошибочного поведения с объяснением
Self-correction: автоматическое исправление ошибок в LLM-коде (до 3 попыток)
Память:

Краткосрочная: общий словарь STATE (df_processed, feature_cols, best_model, action_history)
Долгосрочная: файлы best_model.pkl, best_metrics.json, run_history.jsonl
